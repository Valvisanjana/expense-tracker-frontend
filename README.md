# Expense Tracker Frontend

A React-based Expense Tracker application that helps users manage their income and expenses.

## Features

- User Registration
- User Login
- Dashboard
- Add Expense
- View Expenses
- Track Income, Expenses and Savings

## Technologies Used

- React
- React Router
- Axios
- CSS
